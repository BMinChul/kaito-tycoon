# Requirements

## Visuals
- **Style**: 32-bit Pixel Art, Cyberpunk, Neon colors.
- **Camera**: Isometric (Orthographic).
- **Assets**: 
  - Yapybara character (sprites or 3D billboard).
  - Tech/Crypto station props.

## Features
- **Idle Mechanics**: Automated resource generation.
- **Upgrades**: Spend resources to increase generation rate.
- **Web3 Integration**: (Planned) Wallet connection, Token integration.

## Constraints
- **Asset Safety Protocol**: STRICTLY FORBIDDEN from deleting, renaming, overwriting, or modifying any existing files in the assets folder or any subfolders. Read-only access allowed.
- Use `vibe-starter-3d` for physics and character control where applicable.
- Ensure performant rendering for potential mobile web support.
